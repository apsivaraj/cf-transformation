Deploying application to :tgttype.

Perform the following actions:

  1. You can setup environtment variables to make it easy to run the commands listed here:
    • The docker image repository host (and port) that you will use such as docker.io for DockerHub, or .icr.io or other repository hosts. export REPOHOST=docker.io
    • The namespace that you will store this image to. export REPOSPACE=mynamespace :envvar.
  2. Go to the conversion path directory cd :tgtdir.
  3. Generate the docker image:docker build -t :applname. .
  4. Tag the docker image for a docker repository that is accessible to the cluster.docker tag :applname. ${REPOHOST}/${REPOSPACE}/:applname.
  5. Store the docker image into a docker repository that is accessible to the cluster.docker login ${REPOHOST} docker push ${REPOHOST}/${REPOSPACE}/:applname. :deployapp.
